# Exploration Guide Reference

탐험 전략, 퀘스트 우선순위, 팩션 관리, 리프트 대응을 다룹니다.

## 타일 이동 우선순위

`get_available_actions`의 이동 옵션 중 선택 순서:

1. **스토리 타일** (고유 이름 장소) — 항상 최우선 탐험
2. **퀘스트 마커** (활성 퀘스트가 가리키는 곳) — 퀘스트 따라가기
3. **미탐험 타일** — 미지의 곳을 알려진 곳보다 우선
4. **상점 타일** — 골드 > 150 + 장비 공백 있을 때
5. **휴식 지점** — HP < 60%일 때

### 층 진행 원칙
- 현재 층 접근 가능한 타일을 모두 탐험 후 하강
- 보스 방 발견 시 — HP > 70%일 때만 도전
- `get_world_time` 확인 — 시간대별 유리한 활동이 다름

## 퀘스트 전략

### 수락 기준
- 항상 수락 (`get_active_quests` 결과 = 진행할 것)
- 현재 탐험 중인 지역과 겹치는 퀘스트 우선 (동선 효율)
- 보유하지 않은 아이템 요구 퀘스트 → 먼저 인벤토리 확인

### 완료 우선순위
1. 메인 스토리 퀘스트 (최고 XP)
2. 팩션 퀘스트 (평판 상승)
3. 처치 퀘스트 (어차피 전투와 병행)
4. 수집 퀘스트 (아이템 이미 보유 시만)

## 팩션 관리

`get_all_factions` 결과를 주기적으로 확인합니다.

### 전략
- **주력 팩션 1개 선정**: 일관된 퀘스트 완료로 평판 집중 성장
- **타 팩션 중립 유지**: 주력 팩션의 적 팩션 퀘스트는 피하기
- **평판 Tier 3+**: 특수 퀘스트와 독점 상인 해금

### 팩션 친화 행동
- 해당 팩션 NPC를 보호하는 선택지 선택
- 팩션 의뢰 퀘스트 완료
- 적대 팩션 몬스터 처치

## 리프트(Rift) 대응

`check_rift`로 발견한 리프트 처리 기준:

| 리프트 레벨 | 대응 |
|------------|------|
| 1-2 | 입장 — 좋은 전리품, 관리 가능한 위험 |
| 3 | HP > 80% + 좋은 장비일 때만 입장 |
| 4+ | 최대 레벨 도달 전까지 회피 |

## 시간대 효과

`get_world_time` 확인 후 활동 계획:

| 시간대 | 효과 | 최적 활동 |
|--------|------|----------|
| 새벽(Dawn) | 처치 XP +10% | 전투 집중 |
| 낮(Midday) | 상점 재고 풍부 | 장비 구매 |
| 황혼(Dusk) | 강한 적 + 좋은 드롭 | 고위험 고수익 탐험 |
| 밤(Night) | Thief/Ranger 스텔스 보너스 | 해당 클래스면 야간 탐험 |

## NPC 동기 파악

`get_npc_motivation`으로 NPC 의도 파악:
- **우호적 NPC**: 대화 선택지 → 퀘스트/정보 획득
- **중립 NPC**: 거래 가능성 확인
- **적대적 NPC**: 전투 또는 회피
